## Download sources

All sources are available either on hackage and can be downloaded and
build using `cabal-install`:

```bash
cabal update      # update package index
cabal install H   # installs H
```

or using github:

```bash
git clone ...
git co tagname
```

Do not forget to install all prerequisities.
